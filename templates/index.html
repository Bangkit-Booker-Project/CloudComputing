Bangkit Booker

API untuk Book rekomendation based on user preferences

Endpoint

https://bangkit-booker-352402.et.r.appspot.com

Register

  • URL
    • /register
  • Method
    • POST
  • Request Body
    • username as string
    • email as string, must be unique
    • password as string, must be at least 6 characters

  • Response

    {
  "error": false,
  "message": "User Created Successfully"
}

Login

  • URL
    • /login
  • Method
    • POST
  • Request Body

Reset Password
